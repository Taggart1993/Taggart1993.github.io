Pet American Gaming SMP: Adeptus Mechanicus Empire of Man Anarchy Minecraft Server

Discord Server is >Here


The PAG SMP is a modded Minecraft 2b2t alternative; it is intended to give players a modded Society Experiment gameplay with no queue or payment involved. It all started with one question: "What if 2b2t had mods?" We then set out to create a Minecraft Forge 1.12.2 server.
It allows players to build their own universe in an Anarchy Society Experiment-type gameplay, where you can build your own empire, declare war, forge cities, and take over the universe. It currently is ruled by the Empire of Man, a group led by the accounts PetAmerican (a server admin) and WackyGirl (an umbrella account used by multiple people).
It is planning to build a giant Nether Highway, and mine in Space. The server contains only a few mods in its modpack, including Guns, Space, Computers, Immersive Engineering, and more. A download is provided in our Discord above.
